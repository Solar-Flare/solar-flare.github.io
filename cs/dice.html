Dice 1

Date: 09/18/2018

Minimalistic Terminal Game

This simple incremental dice rolling game with an ending is probably my first real C# project.

Random number generation was probabaly my hardest problem in this process. In C#, Random.Next does not generate a new random number, if done in quick succession, as the generation is based on the system-supplied seed value that is the same value in very similar time intervals. So I would have cases where rolling 5 cubes end up being 30 (6, 6, 6, 6, 6), and I didn't know how to solve it until looking into the documentation.

Overall, though, it still was a relatively fun project that showed me how C# syntax works.
