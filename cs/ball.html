Roll a Ball

Date: 09/02/2018

Roll Roll Roll your Ball

This is an exact replica of the roll a ball tutorial found here on the unity website.
Which means this is a C# Unity project! All that occurs is that the ball can be controlled by the arrow keys, which adds an acceleration instead of affecting the speed much like realistic ball rolling. The cubes add a point, and if all are collected the message 'you win' appears.
This was pretty fun to follow along and as this was directly followed from unity there were zero problems that occurred throughout.
