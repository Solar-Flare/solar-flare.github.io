Image Classifier

Date: 09/16/2018

Link Here

There exists very few scripts that come built in with an image classifier. ml5js is one of them, and I decided to create a html program that takes an input image, sends it to ml5js, and display the returned values.

A couple of problems were incurred throughout the process, namely one that caused the image to not be sent to the server. It turns out the id values were incorrect; only after multiple trials did I realize the problem wasn't with the ml5js framework but rather the process of delivering the image to analyze.

getElementById looks for id of an html element, not the class name. This was causing the communications error.
Besides that, however, no major error was incurred.
