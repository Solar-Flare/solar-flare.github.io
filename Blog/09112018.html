65,537 and the Interesting Number Paradox

Date: 09/11/2018

65,537 is a pretty interesting number. It is currently the largest Fermat prime number known, which is a prime number that also happens to be a Fermat number which is a number expressed in the form 2^(2^n) + 1. If n is 4, then 2^4 is 16, and 2^16 is 65536, and plus one is 65,537. No other fermat numbers are currently known to be prime, which makes this number interesting.

The largest n-gon constructible by a compass and ruler where n is odd also turns out to be 65537.

But the concept of 'interesting' depends wildly on perspective. Some may say 65,537 is not a very interesting number and there are more interesting ones (Like 1729). So there may exist varying degrees of 'interesting'.

But, by definition ALL numbers must be interesting.
If 2 were the smallest unintersting number, that itself would make it interesting. Similarly, if there existed a set of uninteresting numbers, the first number of that list would become interesting by virtue of being the first uninteresting number. In other words, a set of uninteresting numbers cannot exist.

In other words we should appreciate mathematics and every number that comes out of it because by sheer definition all numbers are interesting by default. This is the Interesting Number Paradox.
